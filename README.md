# Project 3: Spots

### Project 3: Spots

Spots is a photo-sharing website where users can:

- create a profile with their avatar, name, and a description of themselves
- upload photos and name them
- have their photos liked by guests

**Technologies and Techniques Used**

This is my first project created without a .pdf brief.
All of the coding was written from scratch using a Figma design.
It is also my first project employing responsive design, which really excited me.
